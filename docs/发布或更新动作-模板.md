# 发布动作模板 

## 目的 
本文件定义一个可交给 AI 的提示词模板，用于生成可直接发布到 Quicker 动作库（分享方式固定为“发布到动作库”）的配置 JSON。自动决定并填充，最终输出一个完整的动作 JSON。

## 置（模板）
```json
{
  "分享方式": "发布到动作库",
  "名称": "XX",
  "说明": "XX",
  "关键词": "XX,XX,填XX",
  "更新说明": "1. XX\n2. XX"
}
```
