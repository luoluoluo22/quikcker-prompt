# quikcker-prompt

为了方便遵循quikcker规范进行动作开发，将一些需要遵循的规范进行说明。

开发的流程是：

1. 编写C#代码
2. 运行测试
3. 测试通过
4. 准备动作定义
5. 准备发布文档
6. 发布
7. 录制操作说明
8. 接受反馈
9. 代码修改
10. 更新动作
